# 校园供水系统智能管理系统

## 项目简介
CampusFlowMonitor 是一个基于Python的校园供水系统智能监控与分析平台。本系统通过对校园水表数据的实时采集与分析，实现用水规律识别、漏损检测和用水效率评估，为校园水资源管理提供数据支持与决策依据。

## 核心功能
🔍 1. 水表关系模型分析
层级关系分析：识别一级与二级水表的用水关系

时间序列分析：支持15分钟、6小时、1天等多时间粒度分析

误差率计算：计算不同层级水表间的用水量误差

趋势可视化：自动生成用水量变化趋势图

🚨 2. 管网漏损检测
异常用水识别：基于连续时段用水一致性检测潜在漏损

漏水率计算：量化评估各水表漏损程度

40404T专项分析：针对特定水表的深度监控

漏损排名：生成漏损率Top-N水表列表

🏫 3. 功能区用水分析
区域划分：将校园划分为宿舍、食堂、教学等七大功能区

用水规律分析：识别不同功能区的用水高峰与低谷

季节模式分析：分析不同季节的用水变化规律

教学活动关联：关联用水数据与学期、假期等教学活动
