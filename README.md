# 校园供水系统智能管理系统

## 项目简介
基于Python的校园供水系统数据分析系统，包含关系模型构建、漏损分析和功能区用水规律分析等功能。

## 功能特性
1. **关系模型分析** - 分析一级和二级水表的用水关系
2. **漏损分析** - 检测供水管网的漏水情况
3. **功能区分析** - 分析不同功能区的用水规律

## 安装要求
- Python 3.7+
- 依赖包：见`requirements.txt`

## 使用方法
方法1：双击 run.py
方法2：命令行运行
python run.py
